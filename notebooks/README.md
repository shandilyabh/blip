# Notebooks

This folder contains Jupyter notebooks and their corresponding PDF exports covering various quantitative finance and machine learning topics:

## Contents

- **DLfAP_AQR. ipynb / DLfAP_AQR.pdf** - Deep Learning for Asset Pricing and Quantitative Research
> AQR, 2020 Paper: https://www.aqr.com/-/media/AQR/Documents/AQR-Insight-Award/2021/Deep-Learning-in-Asset-Pricing.pdf?sc_lang=en
- **MonteCarloPricing_BayesianOptimisation.ipynb / MonteCarloPricing_BayesianOptimisation.pdf** - Monte Carlo Pricing with Bayesian Optimization techniques
> Bayesian Optimisation: https://distill.pub/2020/bayesian-optimization/
> Monte Carlo for Option Pricing: https://intro.quantecon.org/monte_carlo.html
- **Regressor.ipynb / Regressor.pdf** - Regression model implementation and analysis

Each notebook has an associated PDF version.
